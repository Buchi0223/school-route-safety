# 通学路安全確認デモアプリ モバイル版UI改良

**文書バージョン**: 1.0  
**作成日**: 2026年1月16日  
**ステータス**: ドラフト


以下の点についてモバイル版アプリのUIに改良を加える。

## UIデザイン全般
* 地図以外にメニューを帯状に配置し画面上部に設定
* メニューは、経路検索、通学路探検モード、アイコン説明
* 各メニューを選択すると、対応する機能をオーバーレイ表示
* 経路検索は、すでに構築した経路検索機能を動作させる

## 通学路探検モード
* このモードは下記ストーリーに沿って、保護者が子供に通学路に潜む交通上の危険について一緒に考えるものである

### ストーリー
1. 通学路の経路インプット。
　ここでは、経路検索機能と同様の機能によって子供の通学路に沿って地図をタップしながら経路設定する。
　経路設定完了後、経路検索用の窓はオーバーレイ状態から消えて次の工程に移る。

2. 経路ツアーの開始
　先に設定した通学路経路に沿ってGoogle street viewを使ったバーチャルツアーが開始する。ツアーの開始ボタン、ツアーの時間スライドバー、スピード設定用スライドバーの表示はスマホ画面の下部に最小限のサイズで表示する。
　経路上に危険スポットがあれば、そこで一旦停止する

3. 危険について考える
　一旦停止すると、Google stret viewを見ながら保護者が子供にどのような危険があるかを一緒に探す。ガイドとして画面の左下からキャラクターに吹き出しを添えて、セリフを表示する。
　「ここには、どんな危ない場所がひそんでいそうかな？3つ探してみよう！」
タップすると次の説明を表示する。
　「見つけた危ない場所で身を守るにはどうしたらいいかな？」
タップすると、次のセリフとともに吹き出しが消えてツアーが再開
　「続きの道を進もう！」

4. 学校に進むまで繰り返す
　2．と3.の工程が経路を完了するまで繰り返される。
